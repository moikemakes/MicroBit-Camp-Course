# 🐶 Hero Pet – Project Summary

## 📌 Project Overview
The **Hero Pet** project is an interactive, step-by-step coding lesson designed for students aged 9–12. Students use block-based coding to bring a virtual pet to life on the BBC Micro:bit. By the end of the course, they will have a fully functioning digital companion that responds to buttons, movement, and touch, complete with a mini feeding game.

---

## 🎯 Learning Objectives
Through this project, students will learn and practice the following core computing concepts:

1. **Hardware & Inputs:** 
   - Understanding how physical inputs (A/B buttons, shake sensor, touch logo) map to digital outputs.
2. **Sequencing & Drawing:** 
   - Using the LED matrix to plot coordinates and design pixel art (the pet's face).
3. **Loops (Iteration):** 
   - Utilizing `forever` loops to create continuous, repeating animations (blinking/breathing).
4. **Event-Driven Programming:** 
   - Triggering specific code blocks based on user actions (e.g., `on button A pressed`, `on shake`).
5. **Variables:** 
   - Creating, initializing, and updating a variable (`Score`) to track progress in a mini-game.

---

## ✅ Success Criteria
We will know the students have succeeded when they can demonstrate the following:

- [x] **Basic Output:** The Micro:bit successfully displays a custom static pet face on start.
- [x] **Animation:** The pet has at least a 2-frame animation running inside a `forever` loop.
- [x] **Interactivity:** The pet changes its expression/emotion when Button A, Button B, or Button A+B are pressed.
- [x] **Sensors:** The pet responds to a physical shake by waking up and/or playing a sound.
- [x] **Game Logic:** The student has created a `Score` variable that starts at 0 and successfully increases by 1 when the pet is "fed".
- [x] **Extension (Optional):** The student completes at least one Challenge Task (e.g., adding an Angry emotion, implementing secret inputs, or improving the animation).

---

## 📖 Lesson Breakdown  
* **Step 1: Meet the Microbit** – Introduction to the hardware.
* **Step 2: Make Your Pet Face** – Using LEDs to draw the character.
* **Step 3: Animate Your Pet** – Using loops and pauses for animation.
* **Step 4: Add Pet Feelings** – Using button inputs to trigger emotions.
* **Step 5: Smart Pet Upgrade** – Using the accelerometer/shake sensor and sounds.
* **Step 6: Feed Your Pet Game** – Introducing variables to track a score.
* **Step 7: Showcase Time!** – Peer review and reflection.
