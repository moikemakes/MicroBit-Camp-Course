# ✨ Step 3: Animate Your Pet

## 🌟 What's New?
A real pet moves around, right? We are going to use a `forever` loop to make our pet blink or smile continuously!

## 🛠️ What to Do
1. Grab another `show leds` block from the `Basic` toolbox.
2. Put it right below your first face block.
3. Draw the face again, but change the eyes so they look closed!
4. Grab a `pause (ms)` block from the `Basic` toolbox and put it between the faces to slow down the blinking.
5. Put all these blocks inside the `forever` block so your pet blinks all the time!
