# 💖 Step 4: Add Pet Feelings

## 🌟 What's New?
Pets have feelings too! We are going to use the **Input** blocks to change how our pet feels when we press buttons.

## 🛠️ What to Do
Let's add three feelings: Happy, Hungry, and Sleepy!

1. **Feeling Happy 😊:**
   * Go to the `Input` toolbox and grab the `on button A pressed` block.
   * Add a `show icon` block inside and pick the Happy Face!
2. **Feeling Hungry 🤤:**
   * Grab the `on button B pressed` block.
   * Add a `show leds` block and draw an open mouth for food!
3. **Feeling Sleepy 😴:**
   * Grab another `on button A pressed` block. Click the little arrow next to "A" and change it to "A+B".
   * Add a `show icon` block and pick the Sleeping Face!
